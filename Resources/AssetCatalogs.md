# Asset catalogs

Asset catalogs are not supported by GNUstep. In source format, they're bundles with extension `.xcassets`, which are compiled into `.car` files by the Asset Catalog Compiler (part of Xcode).

* [Asset Catalog Help (2013)](https://web.archive.org/web/20140603052310/https://developer.apple.com/library/ios/recipes/xcode_help-image_catalog-1.0/chapters/RevisionHistory.html#//apple_ref/doc/uid/TP40013303-CH12-DontLinkElementID_10)
* [Create asset catalogs and sets (2020)](https://help.apple.com/xcode/mac/current/#/dev10510b1f7)
* [Asset management (new)](https://developer.apple.com/documentation/xcode/asset-management)
* [Asset Catalog Format Reference (2018)](https://developer.apple.com/library/archive/documentation/Xcode/Reference/xcode_ref-Asset_Catalog_Format/index.html)
  * [Color Set type](ColorSetFormat.md)
